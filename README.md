# M-Shop React

A modern and responsive e-commerce web app built with React and Tailwind CSS.

🔗 **Live demo:** [m-shop.netlify.app](https://main--beamish-chaja-45a744.netlify.app/)

---

## Tech Stack

- **React 18** — component-based UI
- **React Router 6** — client-side routing
- **Tailwind CSS 3** — utility-first styling
- **Vite** — fast development build tool
- **Fake Store API** — product data

## Features

- Browse products and filter by category
- Search products by name in real time
- View product detail in a side panel
- Add and remove products from the cart
- Place orders and view order history
- Fully responsive — mobile, tablet and desktop

## Getting Started

```bash
git clone https://github.com/marlonsalazarg/ecommerce-react.git
cd ecommerce-react
npm install
npm run dev
```
